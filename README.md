## Run Locally

Clone the project

```bash
  git clone git@github.com:neriheredia/ecommerce-tecnica.git
```

Go to the project directory

```bash
  cd ecommerce-tecnica
```

Install dependencies

```bash
  pnpm install
```

Start the server

```bash
  pnpm run dev
```
